# mazentalsam.com — Personal Portfolio

Personal portfolio website for **Mazen Talsam**, Back-End Developer and CS student at Wilfrid Laurier University.

Live at [mazentalsam.com](https://mazentalsam.com)

---

## About

A fully custom portfolio site built from scratch — no frameworks, no templates. Designed with a two-column sticky layout, split-curtain intro animation, and a focus on clean, fast, minimal UI.

## Features

- Split-curtain intro sequence with monogram reveal and progress bar
- Sticky two-column layout (profile card + scrollable content)
- Bold split typography with scroll-triggered clip-path animations
- Custom orange cursor with lagging ring
- 3D card tilt effect on hover
- Project and blog post modals
- Contact form connected to Google Sheets
- Fully responsive

## Tech Stack

- HTML, CSS, Vanilla JavaScript
- Font: [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)
- Icons: Font Awesome
- No build tools, no frameworks
